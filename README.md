
# HR Attrition Analysis Dashboard | SQL + Tableau

## Project Overview
This project analyzes employee attrition data to identify key factors affecting employee turnover. Using SQL for data analysis and Tableau for visualization, the project provides actionable insights into workforce trends.


## Objectives
- Analyze employee attrition patterns
- Identify key drivers of employee turnover
- Build an interactive dashboard for HR decision-making


## Tools & Technologies
- SQL (Data Analysis)
- Tableau (Dashboard & Visualization)
- Excel/CSV (Dataset)


##  Dataset
- HR Employee Attrition Dataset
- Contains employee details such as age, department, salary, job role, and attrition status


## Dashboard Features
- KPI Metrics:
  - Total Employees
  - Employees Left
  - Attrition Rate
  - Over time
  
- Attrition Analysis by:
  - Department
  - Job Role
  - Work-Life Balance
  - Overtime
  - Salary
  - Years at company
  
- Interactive Filters:
  - Department
  - Job Role
  - Gender



## Key Insights
- The overall attrition rate is 16.12%, indicating moderate employee turnover.
- Research & Development and Sales departments experience the highest attrition.
- Sales Executive and Research Scientist roles have the highest employee exits.
- Employees working overtime are more likely to leave the organization.
- Lower salary levels are associated with higher attrition rates.
- Poor work-life balance significantly impacts employee retention.
- Most employees leave within the first 5 years at the company.




## 🚀 Conclusion
This project highlights key factors influencing employee attrition and provides insights that can help organizations improve employee retention strategies.
